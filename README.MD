Apenas um jogo simples, usando OOP, concorrência e GUI

O jogador 1(viatura) deve pressionar a tecla 'a' para mover o carro, até o alcançar o outro carro

O jogador 2(carro branco) deve pressionar a tecla 'seta direita' para mover o carro, até alcançar o final da tela

Se o jogador 1 alcançar o outro jogador, antes que o mesmo chegue ao final, ele vence
Se o jogador 2 alcançar o final da tela, ele vence
